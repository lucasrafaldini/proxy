## **MiddleMan**


### **Quick Run**

Setup you local .env file:

```bash
$ cp .env.example .env
```

Use docker-compose to run the project:

```bash
$ docker-compose up
```


### **Testing**

To run unit testing, you can use the script:

```bash
$ ./scripts/test.sh
```

### **Formatting**

To run linters and formatters, you can use the script:

```bash
$ ./scripts/lint.sh
```
